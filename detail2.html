지금까지 배웠던 HTML을 기록해보는 포스트를 남겨볼까 한다. 일단 Github Repositories는 본문 최하단에 첨부하겠다. 난 약 한달 동안 프론트엔드를 교육받았었고 React를 활용해 약 두달간 여러 프로젝트를 진행했던 경험이 있어 html은 그냥 편하게 지나가야 겠다. 라고 생각했었다. 하지만 이 생각은 내 큰 착오였고 그렇게 많이 만지던 코드들인데 기억조차 잘 나지 않았다. 다시 마음을 다 잡고 차근차근 모든 태그들을 사용하고 기록하였다. 이렇게 정리하니 나도 모르게 머릿속에 태그들이 저장되어갔다. 항상 vscode 자동완성 기능만 믿고 대충 공부하던 지난날들을 반성하게 되는 계기가 된 것 같다. https://github.com/HyperQuanx/ChunjaeHtmlStudy
